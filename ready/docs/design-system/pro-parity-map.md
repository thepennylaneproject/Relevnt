# Ready Design System - Pro Tier Parity Map

Ready uses Relevnt's Pro tier as its default visual identity.

## Color Tokens

| Token        | Light Mode | Dark Mode | Usage                |
| ------------ | ---------- | --------- | -------------------- |
| `--bg-1`     | #E8EBE8    | #0C0F12   | Page background      |
| `--bg-2`     | #D9DED9    | #1A2024   | Section background   |
| `--panel-1`  | #E8EBE8    | #1A2024   | Card background      |
| `--accent-1` | #4E808D    | #4E808D   | Primary actions      |
| `--accent-2` | #81B5B6    | #81B5B6   | Highlights           |
| `--accent-3` | #A8C3A2    | #A8C3A2   | Ready/success states |

## Ready State Colors

- **Ready**: `--accent-3` (#A8C3A2) - Sage green
- **In Progress**: `--color-warning` (#F59E0B)
- **Not Started**: `--color-text-muted`
- **Blocked**: `--color-error` (#EF4444)

## Typography

- **Primary Font**: Inter
- **Display Sizes**: 72px (hero), 48px (h1), 32px (h2), 24px (h3)
- **Body**: 18px, 16px, 14px (caption)

## Components

1. **StatCard** - Metrics display with trend indicators
2. **FeatureCard** - Feature tiles with status (A, B, C variants)
3. **Card** - Generic container (default, elevated, outlined)
4. **Button** - Primary, secondary, ghost, danger variants
