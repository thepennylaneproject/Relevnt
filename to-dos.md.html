# Relevnt Roadmap / V2 Wishlist


## Matching & Personas

- Multiple resumes per user with selectable “persona”:

  - Example: Tech Strategy, Beauty Industry, Education Leadership

  - Each persona stores:

    - Attached resume(s)

    - Target titles

    - Include / exclude keywords

    - Preferred locations

    - Salary expectations

    - Remote preference

    - Match weight sliders (salary vs location vs mission vs growth)

- Ability to switch active persona in the UI and re-run the feed instantly

- Saved “match profiles” tied to subscription tiers


## Preferences & Control

- User facing “Relevancy Tuner”:

  - Sliders for salary importance, location importance, remote, mission, growth

  - Clear tooltips that explain what each slider does in plain language

- Saved searches:

  - Nameable searches (for example “High paying remote UX”, “Seattle IC roles”)

  - One click re-run

- Smart defaults by role or industry


## Jobs & Applications

- Employer site detection:

  - Prefer direct employer apply URL when present

  - Badge in UI: “Apply on employer site” vs “Apply via board”

- Basic auto apply rules (for later tiers):

  - Only auto apply if match score above X

  - Only auto apply when salary above Y

  - Only auto apply for remote roles or specific locations


## Education & Transparency

- “Why this job?” explainer on each card:

  - Show the top reasons used in the match score

  - Link to “What Relevnt looks for” mini-guide

- “State of the job market” panel:

  - Pull key stats from the research report into user friendly copy

  - Explain ATS, AI screening, referrals, and how Relevnt works with that


## Feels & Experience

- Multiple personas shown as chips or tabs in the feed

- Onboarding wizard that helps users set:

  - Target titles

  - Locations

  - Salary ranges

  - Work style

- Tier hooks:

  - Free: one persona, one resume, basic feed

  - Pro: multiple personas, saved searches

  - Premium: auto apply rules, deeper analytics
